## 1.基础语法

### 1.基本语法

- 大小写敏感
- 类名使用大驼峰
- 方法名使用小驼峰
- 源文件名与类名相同
- 主方法入口：所有的java程序由public static void main(String[] args)方法开始执行

### 2.java标识符

- 所有标识符都是以字母，美元符（$），或下划线_开始，首字符后可以是字母，美元符，下划线及数字。
- 关键字不能用作标识符

### 3.java修饰符

- 访问控制修饰符 : default, public , protected, private
- 非访问控制修饰符：final,abstract,static,synchronized

### 4.java关键字

- 访问控制：private、protected、public、default
- 类，方法，变量修饰符：abstract、class、extends、final、implements、interface、native、new、static、strictfp、synchronized、transient、volatile
- 程序控制语句：break、case、continue、do、else、for、if、instanceof、return、switch、while
- 错误处理：assert、catch、finally、throw、throws、try
- 包相关：import、package
- 基本类型：boolean、byte、char、double、float、int、long、short
- 变量引用：super、this、void
- 保留关键字：goto、const
- java中的null不是关键字，字面常量，不能用作标识符

## 2.java类与对象

- 类：定义对象的蓝图，包括属性，方法
- 对象：类的实例，具有状态和行为
- 封装：将对象的属性私有化，方法公开
- 继承：一个类可以继承另一个类的属性和方法，形成“是-a”的关系，使用extends关键字，只支持单继承
- 多态：同一接口在不同对象上表现不同的行为
  - 编译时多态：方法重载-同一个类中方法名相同，但参数不同
  - 运行时多态：父类引用指向子类对象，调用重写方法调用子类实现

### 1.构造方法

- 每个类都有构造方法
- 没有显示定义，java编译器会给该类提供一个默认的构造方法
- 创建一个对象的时候，至少要调用一个构造方法，构造方法的名称必须与类同名，一个类可以有多个构造方法。

## 3.java基本数据类型

### 1.内置数据类型

- byte：

  - byte数据类型是8位，有符号的，以二进制补码表示的整数
  - -128（-2^7）~ 127（2^7-1）
  - 默认值0
  - byte 类型用在大型数组中节约空间，主要代替整数，因为 byte 变量占用的空间只有 int 类型的四分之一；

- short:

  - short数据类型是16位，有符号的，以二进制补码表示的整数
  - -32768（2^15） ~ 32767 （2^15-1）
  - 默认值0
  - 一个short变量是int型变量所占空间的二分之一 ，因此也可以用来节省空间

- int:

  - int数据类型是32位，有符号的，以二进制补码表示的整数
  - -2,147,483,648（2^31）~  2,147,483,647（2^31 - 1） 
  - 整型变量的默认类型为int类型
  - 默认值0

- long:

  - long 数据类型是 64 位、有符号的以二进制补码表示的整数；
  - -9,223,372,036,854,775,808（-2^63） ~ 9,223,372,036,854,775,807（2^63 -1） 
  - 主要使用在需要比较大整数的系统上 
  - 默认值是 0L

- float:

  - float 数据类型是单精度、32位、符合IEEE 754标准的浮点数；
  - float 在储存大型浮点数组的时候可节省内存空间；
  - 默认值是 0.0f；
  - 浮点数不能用来表示精确的值，如货币；

- double:

  - double 数据类型是双精度、64 位、符合 IEEE 754 标准的浮点数；

  - 浮点数的默认类型为 double 类型；

  - double类型同样不能表示精确的值，如货币；

  - 默认值是 0.0d；

    >double   d1  = 7D ;
    >double   d2  = 7.; 
    >double   d3  =  8.0; 
    >double   d4  =  8.D; 
    >double   d5  =  12.9867; 

- boolean:

  - boolean数据类型表示一位的信息 
  - 只有两个取值：true 和 false；
  - 默认值是 false；

- char:

  - char 类型是一个单一的 16 位 Unicode 字符；
  - 最小值是 \u0000（十进制等效值为 0） 
  - 最大值是 \uffff（即为 65535）；
  - char 数据类型可以储存任何字符；

### 2.引用类型

- 在Java中，引用类型的变量非常类似于C/C++的指针。引用类型指向一个对象，指向对象的变量是引用变量。这些变量在声明时被指定为一个特定的类型，比如 Employee、Puppy 等。变量一旦声明后，类型就不能被改变了 
- 对象、数组都是引用数据类型。
- 所有引用类型的默认值都是null。
- 一个引用变量可以用来引用任何与之兼容的类型。
